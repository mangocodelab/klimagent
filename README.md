# KlimAgent

A command-line AI agent for code understanding and generation.

## Features

- Natural Language Processing with NVIDIA NIM
- Code interpretation and refactoring
- Interactive debugging session

## Installation

1. Clone the repository:
```git
git clone https://github.com/simular-ai/Agent-S.git
```

2. Install dependencies:
```bash
npm install
```

3. Run the application:
```bash
python3 run_game.py
```

## Usage

1. Request understanding of a function from the standard input.
2. Request generation of a new function based on a prompt.
